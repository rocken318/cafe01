# Café Étoile — Landing Page

A fictional luxury café landing page built with modern web technologies.

## Live Preview

Clone and run locally:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack

- **Next.js 16** — App Router, TypeScript
- **Tailwind CSS v4** — CSS-based theme configuration
- **Framer Motion** — scroll-triggered animations
- **Lucide React** — icons
- **Google Fonts** — Cormorant Garamond + DM Sans

## Sections

| Section | Description |
|---|---|
| **Hero** | Full-viewport with Ken Burns scale + staggered fade-in |
| **Concept** | Asymmetric 5-col grid, 3 philosophy pillars |
| **Menu** | 4-card dark grid with hover animations |
| **Gallery** | Masonry-style photo grid with caption overlays |
| **Info & Footer** | Hours, SVG map placeholder, Instagram link |
